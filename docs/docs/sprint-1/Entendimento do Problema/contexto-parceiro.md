## Contexto do Projeto

&emsp;O projeto foi desenvolvido em parceria com a Píer Seguradora com foco na investigação de soluções tecnológicas para apoio à recuperação de veículos roubados ou furtados.

&emsp;A proposta consiste na utilização de drones integrados a técnicas de visão computacional para realizar a captura e identificação automática de placas veiculares em regiões estratégicas. A partir dessas informações, o sistema realiza consultas em uma base de dados de sinistros para identificar possíveis veículos com registro de roubo ou furto.

&emsp;A iniciativa busca explorar como tecnologias de automação, inteligência artificial e sistemas embarcados podem contribuir para ampliar a capacidade operacional da seguradora, reduzindo limitações presentes no modelo tradicional de busca realizado atualmente.

&emsp;Além do desenvolvimento técnico da solução, o projeto também envolve a análise de viabilidade operacional, restrições regulatórias, limitações tecnológicas e impactos relacionados à privacidade e uso de dados.

---

## Objetivo Geral

&emsp;O objetivo geral do projeto é investigar e desenvolver uma prova de conceito capaz de apoiar o processo de recuperação veicular por meio da integração entre drones, visão computacional e sistemas de consulta de sinistros, buscando aumentar a eficiência da identificação de veículos roubados ou furtados.

---

## Escopo da Prova de Conceito

&emsp;A solução proposta contempla o desenvolvimento de um MVP (Minimum Viable Product) focado em:

- Captura de imagens de veículos utilizando drones;
- Identificação automática de placas veiculares;
- Registro de geolocalização;
- Consulta automatizada em uma base mock de sinistros;
- Geração de alertas para possíveis correspondências identificadas.

&emsp;Por se tratar de uma prova de conceito acadêmica, o projeto não contempla integração com sistemas policiais, utilização de dados reais de clientes ou operação em ambiente urbano real.
