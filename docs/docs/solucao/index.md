# Solução

Este espaço reúne o resumo da solução final desenvolvida pelo **Grupo 03 — Turma 16**.

## Sobre o Projeto

O projeto entrega um MVP para operação de drones em campo com apoio de visão computacional e interface web. A solução centraliza no dashboard o controle do DJI Tello, o acompanhamento por vídeo ao vivo, a telemetria e os fluxos de apoio ao gestor local e ao gestor remoto.

Ao longo das sprints, a proposta evoluiu para uma arquitetura híbrida: o frontend e o backend organizam a operação e a experiência do usuário, enquanto a camada local de borda mantém o que depende de baixa latência, conexão com o drone e processamento de visão computacional. Isso inclui controle de voo, captura e tratamento de imagens, OCR/ALPR e sincronização dos dados gerados em campo.

Na versão final, o MVP ficou pronto para uso operacional, com onboarding para os perfis de gestão, controle de voo pelo próprio dashboard, validação com usuários e uma estrutura mais modular para manutenção e evolução do sistema.
